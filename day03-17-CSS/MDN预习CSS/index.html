这一次我们选择广东的海陵岛

一半是蔚蓝大海🌊一半是椰树林🌴

有晒不完的阳光还有踏不完的浪🏄🏻‍♀️

在一眼望不到头的大海边上,享受着阳光沙滩☀️

具体信息

🏖️名字：广东·海陵岛 📍位置：广东省-阳江市-江城区-闸坡镇-海陵岛 🚗交通：跨海大桥直达 🎫门票：¥0 ⏰游玩时间：两天一夜

路线

🚄动车：广州南-阳江站/15分钟一趟 🚙自驾：高速阳江港下，根据指示牌走省道即可 🚌公交：31路公交，20元/人， 20分钟/班，车程90分钟

打卡点推荐

➡️大角湾：是最早开放的，可以玩水、拍照，玩水上项目，游客相对较多，吃住行比较方便，住宿条件 ➡️十里银滩：沙细人少浪大，有很长的海岸线，日出观赏点，还有海滨栈道 ➡️红树林：看海景建议涨潮时去，赶海建议退潮时 ➡️北洛秘境：百米高空700平米天际泳池，私人海滩，滨海栈道，儿童乐园等
